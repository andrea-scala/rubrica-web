# Rubrica Web

Applicazione web per la gestione di una rubrica telefonica, sviluppata in Python con Flask.

## Requisiti
- Python 3.x
- MySQL

## Installazione
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Avvio
```bash
flask run
```
